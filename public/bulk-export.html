Bulk Flier Export

Follow the steps below to bulk export fliers.

Step 1: Select Communities

Fliers will be populated with logos, urls and access codes from the selected communities.

Step 2: Select Templates

Now, select the templats you'd like to export for. The bulk exporter will generate a PDF of each template selected for each community selected.

You can select existing templates from the list, or create a new template by uploading an image or pasting a CDN URL.
